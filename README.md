# L2J Play Fun Launcher

Автоматична збірка Windows EXE для лаунчера Lineage II сервера L2J Play Fun.

## 🚀 Як це працює

1. Кожен push в `main` — GitHub Actions автоматично збирає `.exe`
2. Реліз створюється автоматично з тегом версії
3. EXE доступний у розділі [Releases](../../releases)

## ✨ Функції (v10)

- **🚀 Паралельне завантаження** — файли викачуються 8 потоками одночасно
- **⚡ Відображення швидкості** — показує MB/s в реальному часі
- **🔍 Очищення зайвих файлів** — при перевірці видаляє локальні файли, яких немає на сервері
- **📊 Індикація завантаження** — під час перевірки чітко показує що саме завантажується
- **🛡️ Жодних зайвих вікон** — результат перевірки відображається тільки в статусі лаунчера
- **🏃 Обробка UAC** — автоматично підвищує права через UAC якщо гра цього потребує
- **📁 Вибір директорії** — лаунчер запам'ятовує шлях до гри
- **🌍 Статус сервера** — онлайн гравців та статус Login/Game серверів
- **🔐 Налаштування IP/порта** — можна змінити адресу сервера

## 📦 Файли

| Файл | Призначення |
|------|-------------|
| `launcher/L2JPlayFun-Launcher.py` | Ісходний код лаунчера (Python 3)
| `launcher/launcher.ico` | Іконка програми |
| `launcher/logo_128.png` | Логотип в GUI |
| `launcher/launcher_api.php` | API для серверного маніфесту |

## 🛠️ Збірка вручну (якщо потрібно)

```bash
cd launcher
pip install pyinstaller
pyinstaller --onefile --windowed --icon=launcher.ico L2JPlayFun-Launcher.py
```

Або через spec:
```bash
pip install pyinstaller
pyinstaller --noconfirm L2JPlayFun-Launcher.spec
```

## 📥 Скачати готовий лаунчер

Перейдіть у [Releases](../../releases/latest) та скачайте `L2J-PlayFun-Launcher.exe`

## ⚠️ Примітка безпеки

Оскільки програма не підписана платним сертифікатом, Windows Defender та Chrome можуть показувати попередження. Це нормально для безкоштовних ігрових лаунчерів. Натисніть «Детальніше» → «Виконати в будь-якому випадку» / «Keep anyway».
